🔧 Firebase Setup Required

This game needs a free Firebase project to sync moves in real time between two players. It takes about 5 minutes to set up:

  1. Go to console.firebase.google.com
  2. Click Add project → name it anything
  3. Click Add app → choose Web (</>)
  4. Copy the firebaseConfig object shown
  5. In the left menu: Build → Realtime Database → Create database → choose test mode
  6. Open this HTML file and paste your config at the top of the <script>

Firebase free tier is more than enough for this — no credit card needed.
